# in-memory cache (Go)

## Для добавления проекта используйте `github.com/denisakhmetovdev/inMemoryCache`
```go
package main

import (
"github.com/denisakhmetovdev/inMemoryCache"
)

func main(){
	// Ваш код
}
```

## Использование пакета
Для создания кеша нужно вызвать метод `New()`


```go
func main(){
	cache1 := cache.New()
}
```
